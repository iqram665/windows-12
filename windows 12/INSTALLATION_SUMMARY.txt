❓ FREQUENTLY ASKED QUESTIONS:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Q: Will my files be deleted?
A: NO. Only registry settings are modified. All your files are safe.

Q: Can I undo the changes?
A: YES. Double-click backup.reg to restore original settings.

Q: How long does it take?
A: Full installation: 20-40 minutes. Quick setup: 2-5 minutes.

Q: Can I use it on multiple computers?
A: YES. Copy the Windows 12 folder to any Windows 11 computer.

Q: Is it safe?
A: YES. All scripts are open-source and only modify system settings.

Q: Do I need internet?
A: For tools installation, YES. For theme/taskbar only, NO.

Q: What if something goes wrong?
A: Import backup.reg to restore. All changes are reversible.

Q: Can I customize further?
A: YES. Edit registry or config.json for advanced customization.

Q: Will it slow down my computer?
A: NO. It actually optimizes performance by disabling telemetry.

📞 TROUBLESHOOTING:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

ERROR: "Access Denied"
→ Solution: Run script as Administrator

ERROR: "Taskbar didn't move"
→ Solution: Restart Windows. Try again.

ERROR: "Script won't run"
→ Solution: Run Launcher.bat instead

ERROR: "Tools not installing"
→ Solution: Check internet. Run Install-Tools.bat separately

ERROR: "Can't import .reg file"
→ Solution: Right-click → "Merge" instead of Open
