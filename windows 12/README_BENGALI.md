# Windows 12 - সম্পূর্ণ ইনস্টলেশন গাইড

## ওভারভিউ
এই প্যাকেজ Windows 11 কে **Windows 12** তে পরিণত করে:
- ✅ Taskbar উপরে স্থানান্তরিত
- ✅ আধুনিক ডার্ক থিম
- ✅ সিস্টেম অপটিমাইজেশন
- ✅ স্বয়ংক্রিয় টুলস ইনস্টলেশন
- ✅ কাস্টম স্টার্টআপ কনফিগারেশন

---

## 📋 সিস্টেম প্রয়োজনীয়তা
- Windows 11 (যেকোনো সংস্করণ)
- Administrator অ্যাক্সেস
- 50GB খালি জায়গা
- ইন্টারনেট সংযোগ

---

## 🚀 দ্রুত শুরু করুন

### পদ্ধতি 1: Launcher ব্যবহার করুন (সবচেয়ে সহজ)

1. **Launcher.bat ফাইলে ডাবল ক্লিক করুন**
2. **মেনু থেকে একটি অপশন নির্বাচন করুন**
3. **Administrator সুবিধা অনুমোদন করুন**
4. **ইনস্টলেশন সম্পন্ন হতে অপেক্ষা করুন**

### পদ্ধতি 2: ব্যাচ ফাইল সরাসরি ব্যবহার করুন

1. **Install-Windows12.bat ফাইলে রাইট ক্লিক করুন**
2. **"Run as Administrator" নির্বাচন করুন**
3. **প্রক্রিয়া সম্পন্ন হতে অপেক্ষা করুন**
4. **কম্পিউটার স্বয়ংক্রিয়ভাবে রিস্টার্ট হবে**

### পদ্ধতি 3: PowerShell স্ক্রিপ্ট ব্যবহার করুন

1. **PowerShell খুলুন (Administrator হিসেবে)**
2. এই কমান্ড টাইপ করুন:
   ```powershell
   cd "f:\Programing code\windows 12"
   .\Install-Windows12.ps1
   ```
3. **ইনস্টলেশন শেষ হতে অপেক্ষা করুন**

## 🎯 ইনস্টলেশন ধাপগুলি

### ধাপ 1: ব্যাকআপ তৈরি
- আপনার রেজিস্ট্রি স্বয়ংক্রিয়ভাবে `backup.reg` এ সংরক্ষিত হয়
- সমস্যা হলে পুনরুদ্ধার করা যায়

### ধাপ 2: Taskbar উপরে সরান
- Taskbar নীচে থেকে শীর্ষে চলে যায়
- রেজিস্ট্রি সেটিংস পরিবর্তন করা হয়

### ধাপ 3: থিম প্রয়োগ করুন
- ডার্ক মোড সক্রিয় করা হয়
- স্বচ্ছতা প্রভাব যোগ করা হয়

### ধাপ 4: সিস্টেম অপটিমাইজ করুন
- অপ্রয়োজনীয় সেবা বন্ধ করা হয়
- স্টার্টআপ সময় কমানো হয়

### ধাপ 5: টুলস ইনস্টল করুন
- Chocolatey প্যাকেজ ম্যানেজার ইনস্টল করা হয়
- জনপ্রিয় ডেভেলপমেন্ট টুলস যুক্ত করা হয়

## ⚙️ Taskbar পিছনে নিয়ে যান

যদি Taskbar উপরে থাকা পছন্দ না হয়:

**কমান্ড প্রম্পটে (Administrator):**
```cmd
reg add "HKCU\Software\Microsoft\Windows\CurrentVersion\Explorer\Advanced" /v TaskbarSide /t REG_DWORD /d 0 /f
```

## ❌ সমস্যা সমাধান

### সমস্যা: "Access Denied" ত্রুটি
**সমাধান:** স্ক্রিপ্ট Administrator হিসেবে চালান

### সমস্যা: Taskbar নড়ল না
**সমাধান:** লগ আউট করুন এবং আবার লগ ইন করুন বা রিস্টার্ট করুন

### সমস্যা: টুলস ইনস্টল হচ্ছে না
**সমাধান:** ইন্টারনেট সংযোগ পরীক্ষা করুন

### পুরানো সেটিংস পুনরুদ্ধার করুন
- `backup.reg` ফাইলে ডাবল ক্লিক করুন
- "Yes" অনুমোদন করুন

## 📊 কী পরিবর্তন হয়েছে

| পরিবর্তন | রেজিস্ট্রি পথ | মূল্য |
|----------|-------------|-------|
| Taskbar | `HKCU\...\Explorer\Advanced` | TaskbarSide = 1 |
| ডার্ক থিম | `HKCU\...\Themes\Personalize` | AppsUseLightTheme = 0 |
| স্বচ্ছতা | `HKCU\...\Themes\Personalize` | EnableTransparency = 1 |

সব পরিবর্তন **উল্টানো যায়** ব্যাকআপ রিস্টোর করে।
